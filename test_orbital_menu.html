🚀 Menu Orbital

Sistema de Navegação Circular 3D - Demonstração Interativa

✨ Funcionalidades Principais

O Menu Orbital é uma experiência de navegação revolucionária que combina design 3D, animações fluidas e interatividade avançada.

Touch & Mouse

Controle total com gestos touch, mouse e roda do mouse para uma experiência fluida em qualquer dispositivo.

Sons Interativos

Feedback sonoro para todas as ações: abertura, fechamento, navegação e interações.

Partículas Animadas

Efeito galáxia com partículas flutuantes que criam uma atmosfera mágica e envolvente.

Rotação 3D

Sistema de rotação orbital com efeitos 3D profundos e animações suaves.

Atalhos de Teclado

Controles rápidos: ESC para fechar, ESPAÇO para toggle rotação, R para resetar.

Totalmente Responsivo

Adapta-se perfeitamente a todos os tamanhos de tela, desde mobile até desktop.

🎮 Como Usar o Menu Orbital

  • Clique no botão do cabeçalho (☰) para abrir o menu orbital
  • Arraste com o mouse para girar a órbita dos ícones
  • Use a roda do mouse para rotação mais precisa
  • Toque e arraste em dispositivos touch para rotação
  • Clique em um ícone para navegar para a página correspondente
  • Pressione ESC para fechar o menu rapidamente
  • Pressione ESPAÇO para alternar rotação automática
  • Pressione R para resetar a rotação

🌟 Recursos Especiais

  • Avatar do usuário no centro com status online/offline
  • Partículas animadas criando efeito galáxia
  • Sons suaves para todas as interações
  • Efeitos 3D com profundidade e perspectiva
  • Animações de entrada e saída fluidas
  • Design adaptativo para todos os dispositivos
