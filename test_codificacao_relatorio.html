🔍 Causas Possíveis:
  • Falta de meta charset UTF-8 no HTML
  • Codificação incorreta no envio do arquivo
  • Falta de BOM (Byte Order Mark) para compatibilidade
  • MIME type sem especificação de charset
Soluções Implementadas
✅ Correções Aplicadas:
  1. Meta charset explícito: <meta charset="UTF-8">
  2. Content-Type com charset: <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
  3. Lang attribute: <html lang="pt-BR">
  4. UTF-8 com BOM: html_content.encode('utf-8-sig')
  5. MIME type com charset: mimetype='text/html; charset=utf-8'
  6. Tratamento de caracteres especiais: Escape de HTML entities
🔧 Código Corrigido:
# Antes (PROBLEMA)
html_content.encode('utf-8')
mimetype='text/html'

# Depois (CORRIGIDO)
html_content.encode('utf-8-sig')  # UTF-8 com BOM
mimetype='text/html; charset=utf-8'
